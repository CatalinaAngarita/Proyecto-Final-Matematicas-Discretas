# 🔐 GUÍA DE ACCESO AL PANEL ADMINISTRATIVO - DIANA NAILS

## 📱 CÓMO ACCEDER AL SISTEMA

### 1. ABRE EL NAVEGADOR
En tu celular o computadora, abre el navegador web.

### 2. URL DE ACCESO
Escribe esta dirección en el navegador:
```
http://127.0.0.1:8000/accounts/login/
```

### 3. CREDENCIALES DE ACCESO
```
Usuario: admin
Contraseña: admin123
```

### 4. DA CLIC EN "INICIAR SESIÓN"
Verás el logo de Diana Nails y un formulario de login.

---

## 📊 QUÉ VERÁ DIANA EN SU DASHBOARD

### PANTALLA PRINCIPAL (`/dashboard/`)

**Sección superior:**
- **Título:** "Dashboard"
- **Fecha:** Día actual con formato completo

**Tarjetas de estadísticas:**
- 📅 **Citas hoy:** Número de citas programadas para hoy
- 👥 **Clientas activas:** Total de clientas registradas
- ⚙️ **Servicios:** Cantidad de servicios ofrecidos
- 📉 **Cancelaciones:** Porcentaje de cancelaciones

**Sección de acciones rápidas:**
- ➕ **Nueva clienta:** Crear perfil de nueva clienta
- 💅 **Nuevo servicio:** Agregar nuevos servicios
- 🕐 **Configurar horario:** Definir horarios laborales
- 📊 **Ver estadísticas:** Análisis detallado del negocio

**Gráfico de actividad semanal:**
- Visualización de citas completadas por día
- Lunes a Sábado

**Tabla de próximas citas:**
- Lista de citas pendientes
- Información: Clienta, Servicio, Fecha, Hora, Estado
- Botón "Ver todas" para lista completa

---

## 📋 SECCIÓN DE CITAS (`/appointments/`)

### FUNCIONALIDADES DISPONIBLES:

1. **VER TODAS LAS CITAS**
   - Lista completa con paginación (30 por página)
   - Filtros por fecha y estado
   - Información detallada de cada cita

2. **CREAR CITAS MANUALMENTE**
   - Formulario completo
   - Selección de clienta, especialista, servicio
   - Configuración de fecha y hora
   - Sistema de cálculo automático de hora fin

3. **EDITAR CITAS**
   - Modificar fecha, hora, servicio
   - Cambiar estado
   - Agregar notas

4. **CANCELAR CITAS**
   - Registro de motivo de cancelación
   - Sistema de tracking de cancelaciones
   - Análisis de tendencias

5. **VER DETALLES**
   - Información completa de la cita
   - Historial de cambios
   - Datos de la clienta

---

## 🔄 ESTADOS DE LAS CITAS

- **PENDIENTE:** Cita recién creada
- **CONFIRMED:** Clienta confirmó asistencia
- **IN_PROGRESS:** Servicio está siendo realizado
- **COMPLETED:** Servicio finalizado
- **CANCELLED:** Cita cancelada
- **NO_SHOW:** Clienta no se presentó
